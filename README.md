# Java Collections Benchmark  
## ArrayList vs LinkedList Performance Comparison

Консольное приложение на Java для сравнения производительности коллекций `ArrayList` и `LinkedList`.

Проект измеряет время выполнения базовых операций и выводит результаты в виде таблицы.

---

## Цель работы

Сравнить производительность следующих операций:

- `add`
- `remove`
- `get`
- `clear`

для коллекций:

- `ArrayList`
- `LinkedList`

Каждая операция выполняется заданное количество раз, после чего фиксируется время выполнения.

---

## Тестируемые операции

### add
- Добавление элементов в конец
- Добавление элементов в начало
- Добавление элементов в середину

### remove
- Удаление элементов из начала
- Удаление элементов с конца
- Удаление элементов из середины

### get
- Получение элемента по индексу

### clear
- Очистка всего списка

---

## Подход к измерению

- Используется `System.nanoTime()` для высокой точности измерений
- Перед измерением коллекции заполняются тестовыми данными
- Каждая операция выполняется N раз
- Результаты сохраняются в объект `Result`
- Итоговая таблица выводится через `TablePrinter`
